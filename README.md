# 🍳 家庭小厨 (Home Chef) - 微信小程序

> 一个基于原生微信小程序开发的家庭菜谱应用，解决“今天吃什么”的世纪难题。
> 收录了经典的家常菜及“老乡鸡”风味菜谱，提供详细的烹饪步骤和随机选菜功能。

## ✨ 项目简介 (Introduction)

本项目是一个 **Mobile-first** 的微信小程序，旨在帮助家庭成员快速查看家里会做的菜肴，并提供趣味性的随机选菜功能。

项目由 **Trae (AI IDE)** 辅助开发，采用 Solo 模式从零构建。数据源经过精心整理，包含详细的食材清单和制作步骤，不仅是一个菜单，更是一本掌上烹饪指南。

## 📱 功能特性 (Features)

- **🍱 菜品分类浏览**：支持按“炒菜”、“炖菜”、“主食”、“凉拌”等类别筛选菜品。
- **📖 详细烹饪指南**：每道菜均包含高清图片、所需食材及详细的分步制作教程。
- **🎲 今天吃什么**：内置随机算法，一键生成“三菜一汤”或“一荤一素”组合，治愈选择困难症。
- **🔍 智能搜索**：(可选) 支持通过菜名快速查找食谱。
- **🖼️ 沉浸式UI**：采用暖橙色主调，营造温馨的家庭烹饪氛围。

## 🛠 技术栈 (Tech Stack)

- **平台**: 微信小程序 (WeChat Mini Program)
- **框架**: Native (WXML + WXSS + JavaScript)
- **样式**: Flexbox / Grid 布局，适配各种移动端屏幕
- **数据**: 本地 Mock 数据 (JSON/JS)，基于 Markdown 文件解析生成
- **开发工具**: Trae (AI 辅助编程), 微信开发者工具
