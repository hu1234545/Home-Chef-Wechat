# 🍳 家庭小厨 (Home Chef) - 微信小程序

> 一个基于原生微信小程序开发的家庭菜谱应用，解决“今天吃什么”的世纪难题。
> 收录了经典的家常菜及“老乡鸡”风味菜谱，采用**分包加载**架构，提供详细的烹饪步骤和随机选菜功能。

## ✨ 项目简介 (Introduction)

本项目是一个 **Mobile-first** 的微信小程序，旨在帮助家庭成员快速查看家里会做的菜肴。
项目由 **Trae (AI)** 辅助开发，数据源基于真实菜谱 PDF 解析生成。为了优化性能和突破主包体积限制，项目采用了微信小程序的 **分包 (Subpackages)** 架构。

## 📱 功能特性 (Features)

- **🍱 菜品分类浏览**：支持按“招牌汤类”、“美味荤菜”等类别筛选。
- **📖 详细烹饪指南**：每道菜包含高清大图及详细的分步制作教程。
- **🎲 今天吃什么**：内置随机算法，一键生成“三菜一汤”组合，治愈选择困难症。
- **⚡️ 性能优化**：使用分包加载技术，确保首屏加载速度极快。

## 🛠 技术栈 (Tech Stack)

- **平台**: 微信小程序 (WeChat Mini Program)
- **架构**: 原生框架 + 分包加载 (Subpackages)
- **样式**: WXSS (Flexbox布局)
- **数据**: 本地 Mock 数据 (自动解析 Markdown 生成)
- **工具**: Trae (AI IDE), 微信开发者工具
